# AI/Vibe 开发教程仓库合集

AI 辅助编程正在深刻改变开发工作流。从提示词工程到 Agentic AI，从初学者入门到高级架构设计，GitHub 上涌现了大量优质开源项目。本文精选 11 个具有代表性的仓库，涵盖初学者课程、Agent 架构、效率工具、中文资源等方向，帮助你快速建立 AI/Vibe Coding 的知识体系。

## 初学者入门

**适合人群**
