# 保研准备

保研，全称"推荐优秀应届本科毕业生免试攻读研究生"，简单说就是不用参加考研初试，直接进入复试。听起来很香，但竞争一点也不小。

## 什么是保研

保研是中国研究生招生的一种方式。符合条件的本科生，可以免去考研初试，直接参加目标院校的复试。复试通过后，就能被录取为研究生。

### 保研率

不同学校的保研率差别很大：

| 学校类型 | 保研率 |
|----------|--------|
| 顶尖 985（清北复交） | 30-50% |
| 普通 985 | 15-30% |
| 211 高校 | 10-20% |
| 普通一本 | 5-10% |
| 双非院校 | 1-5% |

保研率越低，竞争越激烈。如果你在一所普通一本，想保研就得从大一开始准备。

### 保研类型

- **学术型硕士**：偏研究，适合想读博或做科研的同学
- **专业型硕士**：偏实践，适合想直接工作的同学
- **直博**：本科毕业直接读博士，时间短但风险大

## 保研条件

### 硬性条件

这些是必须满足的条件，缺一个都不行：

**成绩（GPA）**

这是最重要的条件。一般来说：

- 985 高校：排名前 20-30%
- 211 高校：排名前 15-25%
- 普通一本：排名前 10-20%

GPA 是硬门槛，排名不够的话，其他条件再好也没用。

**英语成绩**

大部分学校要求通过英语四级，有些学校要求六级。六级分数越高越好，500 分以上比较稳妥。

**无挂科**

挂科基本告别保研。即使补考过了，有些学校也不认。

### 加分项

这些条件能让你在保研中更有竞争力：

**竞赛获奖**

| 竞赛 | 加分程度 |
|------|----------|
| ACM/ICPC 区域赛金牌 | 非常高 |
| ACM/ICPC 区域赛银牌 | 高 |
| 数学建模国赛一等奖 | 高 |
| 数学建模国赛二三等奖 | 中 |
| 挑战杯/互联网+省级以上 | 中 |
| 其他国家级竞赛 | 中低 |

**科研经历**

- 发表论文：SCI/EI/核心期刊，第一作者加分最多
- 参与科研项目：国家级、省级项目
- 发明专利：有实际价值的专利

**实习经历**

大厂实习对保研有一定帮助，但不如竞赛和论文重要。

**开源贡献**

GitHub 上有 star 数较多的项目，或者给知名开源项目贡献过代码。

## 保研时间线

### 大一到大三上：积累阶段

这段时间主要是积累硬实力：

- **保持 GPA**：从大一就开始认真对待每一门课
- **参加竞赛**：ACM、数学建模，越早开始越好
- **做科研**：大二开始联系导师，进实验室
- **过六级**：大一或大二把六级过了

### 大三下（3-6 月）：夏令营

夏令营是保研最重要的渠道，大部分名额都在这时候确定。

**3-4 月：准备材料**

- 个人简历
- 成绩单（前 5 学期）
- 排名证明
- 英语成绩证明
- 获奖证书
- 论文/专利
- 推荐信（2-3 封）
- 个人陈述

**5-6 月：投递和参加**

- 关注各学校官网的夏令营通知
- 按要求投递材料
- 参加夏令营活动（通常 3-5 天）

### 大四上（9-10 月）：预推免和正式推免

**预推免（9 月）：**

夏令营没拿到 offer 的同学，预推免是第二次机会。流程和夏令营类似，但时间更短，竞争也可能更大。

**正式推免（10 月）：**

在研招网上填报志愿，确认录取。这是最后一次机会。

## 夏令营准备

### 材料准备

**个人简历：**

- 一页纸，简洁明了
- 突出 GPA、竞赛、科研、实习
- 用数据说话，比如"GPA 3.8/4.0，排名 5/200"

**个人陈述：**

- 说明自己的学术背景、研究兴趣、未来规划
- 为什么选择这个学校/导师
- 1000-1500 字

**推荐信：**

- 找熟悉自己的老师写
- 最好是教授或副教授
- 内容要具体，不要空话

### 面试准备

夏令营面试通常包括以下几个环节：

**英语自我介绍（2-3 分钟）：**

提前准备好，背熟。内容包括：

- 基本信息（姓名、学校、专业）
- 学术背景（GPA、排名）
- 科研/项目经历
- 研究兴趣
- 为什么选择这个学校/方向

**专业知识面试：**

- 复习专业课基础知识
- 重点复习和目标方向相关的课程
- 准备一些常见问题

**项目/科研经历：**

- 能清楚地介绍自己做过的项目
- 说明自己在项目中的角色和贡献
- 准备好回答"遇到了什么困难，怎么解决的"

**常见面试问题：**

- 自我介绍
- 介绍你的项目/科研经历
- 为什么选择我们学校？
- 你的研究兴趣是什么？
- 读过什么论文？
- 未来规划是什么？
- 有什么问题想问我们？

### 面试技巧

- **自信但不自大**：展示自己的优势，但不要吹牛
- **诚实**：不会的问题就说不会，不要瞎编
- **条理清晰**：回答问题要有逻辑，分点说
- **准备问题**：准备几个问面试官的问题，显示你的兴趣

## 联系导师

### 什么时候联系

- 夏令营前 1-2 个月开始联系
- 越早越好，好导师名额有限
- 如果夏令营没联系上，预推免还可以继续

### 怎么联系

**发邮件：**

这是最常见的方式。邮件要简洁、礼貌、有针对性。

**邮件模板：**

```
主题：XXX大学XXX专业学生申请贵课题组研究生

尊敬的XXX教授：

您好！我是XXX大学XXX专业的大三学生XXX，预计GPA排名前X%。

我对您的研究方向非常感兴趣，特别是您在XXX方面的研究。
我阅读了您近期发表的《XXX》，对其中的XXX方法印象深刻。

我在本科期间：
- 参与了XXX项目，负责XXX
- 发表了XXX论文
- 获得了XXX竞赛XXX奖项

我非常希望能够加入您的课题组攻读研究生，不知您是否还有招生名额？

附件是我的简历和成绩单，期待您的回复。

此致
敬礼

XXX
XXX大学XXX专业
邮箱：xxx@xxx.com
手机：xxx
```

**其他联系方式：**

- 学长学姐推荐
- 学术会议认识
- 导师的研究生帮忙引见

### 选导师的注意事项

- **研究方向**：和自己的兴趣匹配
- **导师风格**：是放养型还是严管型，看自己适合哪种
- **课题组氛围**：问问在读的研究生
- **导师资源**：项目多不多、经费够不够、有没有 industry connection

## 保研 vs 考研

### 保研的优势

- 不用准备考研初试，节省大量时间
- 可以提前选导师，选择余地更大
- 时间成本低，心理压力相对小
- 有些学校只接受保研生（如中科院某些所）

### 保研的劣势

- 需要前三年保持好成绩，不能松懈
- 竞争激烈，尤其是好学校
- 本校保研率高，外校保研难度大
- 一旦没保上，准备考研的时间很紧张

### 考研的优势

- 可以跨学校、跨专业，选择范围广
- 不受本科成绩限制，一次考试定胜负
- 准备时间相对集中，不需要三年持续投入

### 考研的劣势

- 备考压力大，时间成本高
- 竞争激烈，尤其是热门学校和专业
- 只有一次机会（初试），发挥失常就没戏

## 一些实用建议

### 大一就开始准备

保研是三年的积累，不是大三才开始的事。从大一就要：

- 认真对待每一门课，保持 GPA
- 参加竞赛，积累获奖
- 找导师进实验室，积累科研经历

### 信息收集很重要

保研的信息差很大。有些同学早早知道哪些学校有夏令营、哪些导师在招人，有些同学到大三才开始了解。

信息渠道：

- 学校官网
- 保研论坛（保研岛、保研人）
- 学长学姐
- 微信群/QQ 群

### 心态要好

保研的过程可能会很焦虑。夏令营被拒、联系导师没回复，这些都是正常的。保持好心态，坚持下去，总会有结果的。
