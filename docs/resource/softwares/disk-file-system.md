# 磁盘与文件系统

磁盘管理和文件系统是 Windows 使用的基础知识。分区、格式化、文件类型，这些操作用对了能让数据更安全，用错了可能丢数据。

## 文件系统

### 快速带你了解常用的文件系统

NTFS、FAT32、exFAT、ReFS，Windows 支持好几种文件系统，每种都有不同的特点和适用场景：

- **NTFS** - Windows 默认文件系统，支持权限控制、加密、压缩，日常使用首选
- **FAT32** - 兼容性最好，但单文件不能超过 4GB，适合 U 盘
- **exFAT** - 跨平台兼容，没有 4GB 限制，U 盘和移动硬盘推荐
- **ReFS** - 新一代文件系统，数据完整性更强，适合服务器
