# 电脑故障排查

电脑点不亮、蓝屏、重启，这些问题很常见。学会排查故障，不用每次都求人。这篇指南把常见故障的排查思路和解决方法都整理好了。

## 点不亮

### 电脑点不亮？黑屏？重启？史上最强电脑问题排查诊断攻略

电脑点不亮是最常见的问题。先检查电源线有没有接好，内存有没有插紧，显卡供电有没有接。主板上的 Debug 灯能指示问题所在，对照说明书看。

<BiliBili bvid="BV1UY411W7Y9" />

### 点不亮的排查流程

电脑点不亮不要慌，按这个顺序一步步排查：

**第一步：检查电源**

1. 电源线有没有插好（墙插和电源端都检查）
2. 电源背面的开关有没有打开（I 那边是开，O 那边是关）
3. 插排有没有开、有没有电
4. 换个插座试试

**第二步：检查供电线**

1. 主板 24pin 供电线有没有插紧
2. CPU 8pin 供电线有没有插紧（主板左上角）
3. 显卡供电线有没有插紧
4. 模组电源两端都要检查（电源端和主板端）

**第三步：检查硬件接触**

1. 内存重新拔插一次（最常见的问题）
2. 显卡重新拔插一次
3. CPU 检查有没有装好（不太可能松，但检查一下）
4. M.2 固态检查有没有插紧

**第四步：最小化测试**

如果以上都不行，做最小化测试：

1. 只留一根内存
2. 拔掉显卡（用核显的话）
3. 拔掉所有硬盘
4. 拔掉所有 USB 设备
5. 只留 CPU + 一根内存 + 电源

能点亮了再一个个加回去，找出是哪个硬件的问题。

**第五步：看 Debug 灯**

大多数主板都有 Debug 灯（或 Debug 数码管），能指示问题所在：

| 灯亮 | 可能问题 | 解决方法 |
|------|---------|---------|
| CPU 灯亮 | CPU 没装好或不兼容 | 重新安装 CPU，检查针脚 |
| DRAM 灯亮 | 内存问题 | 重新插拔内存，换槽试试 |
| VGA 灯亮 | 显卡问题 | 重新插拔显卡，检查供电 |
| BOOT 灯亮 | 启动设备问题 | 检查硬盘连接 |

**第六步：检查显示器**

1. 显示器有没有开、有没有接电源
2. 信号线有没有接对（接显卡，不是主板）
3. 显示器输入源选对了没有（HDMI/DP）
4. 换根线试试

### 有电但点不亮的常见原因

| 现象 | 可能原因 | 解决方法 |
|------|---------|---------|
| 风扇转但没画面 | 内存没插紧 | 重新插拔内存 |
| 风扇转但没画面 | 显卡没插紧 | 重新插拔显卡 |
| 风扇转但没画面 | 显示器线接错了 | 接显卡不是主板 |
| 反复重启 | CPU 过热 | 检查散热器 |
| 反复重启 | 内存频率太高 | 进 BIOS 降频 |
| 反复重启 | 电源功率不够 | 换大功率电源 |
| 有画面进不了系统 | 硬盘没接好 | 检查硬盘线 |
| 有画面进不了系统 | 启动顺序错误 | 进 BIOS 调整 |

### 新装机第一次点不亮怎么办？

新装机第一次点不亮，90% 是以下原因：

1. **内存没插紧**：最常见的原因，重新拔插，用力按到"咔嗒"一声
2. **CPU 供电线没接**：主板左上角的 8pin 线
3. **显示器线接错**：接到了主板而不是显卡
4. **电源没开**：背面开关没打到 I
5. **前面板跳线接错**：开机键线接错了位置

## 蓝屏

### 蓝屏了怎么办？常见蓝屏错误代码及解决方法

蓝屏（BSOD）是 Windows 遇到严重错误时的保护机制。别慌，大多数蓝屏都能解决。

<BiliBili bvid="BV1UY411W7Y9" />

### 常见蓝屏错误代码

| 错误代码 | 中文名 | 常见原因 | 解决方法 |
|---------|--------|---------|---------|
| CRITICAL_PROCESS_DIED | 关键进程终止 | 系统文件损坏 | SFC 修复或重装 |
| MEMORY_MANAGEMENT | 内存管理 | 内存故障或不兼容 | 重新插拔，跑 MemTest |
| IRQL_NOT_LESS_OR_EQUAL | 驱动错误 | 驱动冲突或损坏 | 更新/回退驱动 |
| PAGE_FAULT_IN_NONPAGED_AREA | 页面错误 | 内存或硬盘故障 | 检查内存和硬盘 |
| KERNEL_DATA_INPAGE_ERROR | 内核数据错误 | 硬盘问题 | 检查硬盘，换数据线 |
| SYSTEM_SERVICE_EXCEPTION | 系统服务异常 | 驱动或软件冲突 | 更新驱动，卸载问题软件 |
| DPC_WATCHDOG_VIOLATION | DPC 监视器超时 | 驱动响应太慢 | 更新 SSD 和芯片组驱动 |
| WHEA_UNCORRECTABLE_ERROR | 硬件错误 | CPU/内存/主板硬件问题 | 检查硬件，降低频率 |
| VIDEO_TDR_FAILURE | 显卡驱动超时 | 显卡驱动或硬件问题 | 更新驱动，检查显卡 |
| NTFS_FILE_SYSTEM | NTFS 文件系统错误 | 硬盘文件系统损坏 | 运行 chkdsk 修复 |

### 蓝屏排查步骤

**1. 记录错误代码**
- 蓝屏时拍照记录错误代码
- 事件查看器里也有详细信息：Win + X → 事件查看器 → Windows 日志 → 系统

**2. 最近改了什么？**
- 装了新软件？卸载试试
- 更新了驱动？回退试试
- 加了新硬件？拆掉试试
- 改了 BIOS 设置？恢复默认

**3. 硬件排查**
- 跑 MemTest86 检查内存（至少跑 1 小时）
- 跑 CrystalDiskInfo 检查硬盘健康
- 跑 FurMark 检查显卡稳定性
- 检查 CPU 和显卡温度

**4. 系统修复**
```
# 修复系统文件
sfc /scannow

# 修复系统映像
DISM /Online /Cleanup-Image /RestoreHealth

# 检查硬盘
chkdsk C: /f /r
```

**5. 最后手段**
- 重装系统
- 如果重装还蓝屏，基本是硬件问题

### 蓝屏日志在哪里看？

蓝屏时系统会生成 dump 文件，记录了当时的状态：

1. **小 dump 文件**：`C:\Windows\Minidump\` 目录下
2. **完整 dump 文件**：`C:\Windows\MEMORY.DMP`

用 BlueScreenView（免费工具）可以打开 dump 文件，快速定位是哪个驱动导致的蓝屏。

下载地址：https://www.nirsoft.net/utils/blue_screen_view.html

## 清灰维护

### 2026 全网最细心的笔记本清灰教程

笔记本用久了会积灰，导致散热变差、性能下降。清灰能让性能提升 30%，温度降到 60 度。

<BiliBili bvid="BV1hCPvzyEeT" />

### 台式机清灰教程

台式机也需要清灰，建议每半年到一年清一次。

**清灰前准备：**
- 十字螺丝刀
- 压缩空气罐（或吹风机冷风）
- 软毛刷
- 干抹布

**清灰步骤：**

1. **关机断电**：拔掉电源线，按开机键放掉余电
2. **拆机箱侧板**：拧掉螺丝或按下卡扣
3. **清理风扇**：用压缩空气吹或软毛刷刷
   - CPU 散热器风扇
   - 显卡风扇
   - 机箱风扇
   - 电源风扇（不建议拆电源，从外面吹）
4. **清理防尘网**：拆下来水洗，晾干装回
5. **清理散热鳍片**：用压缩空气吹掉鳍片间的灰尘
6. **检查线材**：有没有松动或破损

**注意事项：**
- 不要用手摸主板上的元件
- 风扇不要用力转，可能损坏轴承
- 压缩空气罐倾斜会喷液体，保持直立
- 清完后检查风扇有没有插回供电线

### 笔记本清灰教程

笔记本清灰比台式机复杂，需要拆机。不建议新手自己操作，可以找售后或维修店。

**自己清灰的步骤：**

1. **查拆机教程**：在 B 站搜你的笔记本型号 + 拆机
2. **准备工具**：螺丝刀、撬棒、硅脂
3. **拆后盖**：拧掉所有螺丝，用撬棒撬开卡扣
4. **断电池**：先断电池排线，防止短路
5. **拆散热模组**：拧掉螺丝，轻轻提起
6. **清理风扇**：用刷子和压缩空气清理
7. **换硅脂**：擦掉旧硅脂，涂新的
8. **装回去**：按拆的反顺序装回

**风险提示：**
- 拆坏了不在保修范围内
- 螺丝长短不一样，装错位置可能拧穿主板
- 排线很脆弱，用力过大会断
- 没有把握就找专业人士

### 硅脂多久换一次？

- **台式机**：2-3 年换一次
- **笔记本**：1-2 年换一次（笔记本散热更紧张）

**什么时候该换硅脂？**
- CPU 温度比以前高了很多（比如以前满载 70°C，现在 90°C）
- 清灰后温度还是高
- 烤机时频繁降频

**硅脂推荐：**
- 性价比：利民 TF7、信越 7921
- 高端：暴力熊液金（需要小心操作）、信越 X-23

**涂硅脂方法：**
1. 用酒精擦掉旧硅脂
2. 新硅脂挤米粒大小在 CPU 中心
3. 装散热器时靠压力自然压平
4. 不要涂太多，溢出来可能短路

## 过热降频

### CPU 温度过高怎么办？

CPU 温度过高会触发保护机制，自动降频以减少发热。表现就是电脑变卡、游戏掉帧。

**正常温度范围：**

| 状态 | 正常温度 | 需要注意 | 过高 |
|------|---------|---------|------|
| 待机 | 30-45°C | 45-55°C | 55°C+ |
| 轻度使用 | 40-60°C | 60-70°C | 70°C+ |
| 满载 | 60-80°C | 80-90°C | 90°C+ |

**温度过高的解决方法：**

1. **清灰**：风扇和散热鳍片积灰是最常见原因
2. **换硅脂**：硅脂干了导热效果变差
3. **检查散热器**：散热器有没有装好、螺丝有没有拧紧
4. **改善机箱风道**：加机箱风扇，确保前进后出
5. **降低 CPU 功耗**：BIOS 里降低 PL1/PL2 功耗限制
6. **换更好的散热器**：原装散热器可能压不住

### 显卡温度过高怎么办？

显卡正常满载温度在 70-85°C，超过 90°C 就需要注意。

**解决方法：**

1. **清灰**：显卡风扇积灰
2. **换硅脂**：显卡硅脂也会干
3. **机箱风道**：确保显卡有足够进风
4. **降低功耗**：MSI Afterburner 降低功耗限制
5. **提高风扇转速**：用 MSI Afterburner 调风扇曲线
6. **显卡支架**：防止显卡弯曲影响散热

## 噪音问题

### 电脑噪音大怎么办？

电脑噪音主要来自风扇和硬盘。

**噪音来源排查：**

| 噪音类型 | 来源 | 解决方法 |
|---------|------|---------|
| 嗡嗡声 | 风扇转速高 | 调低风扇转速曲线 |
| 哒哒声 | 机械硬盘 | 换固态硬盘 |
| 嘎嘎声 | 风扇轴承坏了 | 换风扇 |
| 滋滋声 | 电流声（啸叫） | 正常现象，严重可换电源 |
| 呼呼风声 | 机箱风道不好 | 改善风道，加防尘网 |

**降低风扇噪音：**

1. **BIOS 里调风扇曲线**：低负载时降低转速
2. **换静音风扇**：猫头鹰、Arctic P12 等静音风扇
3. **加减震垫**：风扇和机箱之间加橡胶垫
4. **清理灰尘**：灰尘会让风扇失去平衡，产生噪音

**显卡啸叫（Coil Whine）：**

显卡高负载时发出的滋滋声，是电感振动导致的，不影响性能和寿命。无法完全消除，只能：
- 限制帧率减少负载
- 换电源（可能改善）
- 送售后（严重的可以换新）

## 硬件故障

### 内存故障怎么判断？

内存故障的表现：
- 蓝屏，错误代码 MEMORY_MANAGEMENT
- 开机 Debug 灯 DRAM 亮
- 进系统后频繁崩溃
- 游戏闪退

**排查方法：**

1. **单根内存测试**：一次只插一根内存，找出坏的那根
2. **MemTest86 测试**：制作 U 盘启动盘，开机跑测试
   - 下载：https://www.memtest86.com/
   - 至少跑 1 小时，有错误就是内存问题
3. **换插槽**：排除主板插槽问题
4. **降频测试**：内存频率太高可能不稳定，降频试试

### 硬盘故障怎么判断？

硬盘故障的表现：
- 开机找不到硬盘
- 系统卡顿、文件打不开
- 蓝屏 KERNEL_DATA_INPAGE_ERROR
- 文件莫名消失或损坏

**排查方法：**

1. **CrystalDiskInfo 检查**：
   - 健康状态：良好/注意/警告/危险
   - 重新分配扇区计数：不为 0 说明有坏块
   - 待映射扇区数：不为 0 说明有潜在问题

2. **检查数据线**：换根 SATA 数据线或换 M.2 插槽

3. **跑 CrystalDiskMark**：速度异常低说明硬盘有问题

4. **备份数据**：发现硬盘有问题第一时间备份，别等

### 电源故障怎么判断？

电源故障的表现：
- 电脑随机重启
- 开机偶尔点不亮
- 高负载时死机
- 有烧焦味

**排查方法：**

1. **检查电源功率**：用功率计测量整机功耗，看是否超过电源额定功率
2. **替换法**：借一个好的电源试试
3. **检查线材**：有没有破损、松动
4. **闻味道**：有烧焦味赶紧断电

### 显卡故障怎么判断？

显卡故障的表现：
- 花屏（屏幕上出现彩色条纹或方块）
- 黑屏（有声音没画面）
- 驱动频繁崩溃
- 游戏闪退

**排查方法：**

1. **更新驱动**：先排除软件问题
2. **降频测试**：用 MSI Afterburner 降频，稳定了说明是频率太高
3. **换显示线**：排除线材问题
4. **换显示器**：排除显示器问题
5. **跑 FurMark**：15 分钟内花屏或黑屏，显卡有问题
6. **核显测试**：拔掉独显，用核显开机，能正常说明独显有问题

## 软件故障

### 系统进不去怎么办？

**情况 1：卡在 Windows Logo**
- 强制关机三次，进入自动修复
- 安全模式下卸载最近装的软件或驱动
- 系统还原到之前的还原点
- 最后手段：重装系统

**情况 2：黑屏只有鼠标**
- Ctrl+Alt+Delete 看能不能打开任务管理器
- 任务管理器 → 文件 → 运行新任务 → 输入 `explorer.exe`
- 如果不行，进安全模式排查

**情况 3：无限重启**
- 进安全模式（开机时按 F8 或强制关机三次）
- 卸载最近的更新或驱动
- 系统还原
- 重装系统

### 怎么进安全模式？

**方法 1：设置里进**
- 设置 → 系统 → 恢复 → 高级启动 → 疑难解答 → 高级选项 → 启动设置 → 重启 → 按 F4

**方法 2：强制关机三次**
- 开机时看到 Logo 就强制关机
- 重复三次，系统会自动进入修复模式
- 疑难解答 → 高级选项 → 启动设置 → 重启 → 按 F4

**方法 3：命令行**
```
msconfig
```
引导 → 安全引导 → 确定 → 重启

**安全模式能做什么？**
- 卸载有问题的软件或驱动
- 运行杀毒软件扫描
- 系统还原
- 备份数据

### 怎么修复系统引导？

系统引导损坏会导致进不了系统，修复方法：

**方法 1：自动修复**
- 用 Windows 安装 U 盘启动
- 选择"修复计算机" → 疑难解答 → 启动修复

**方法 2：命令行修复**
```
bootrec /fixmbr
bootrec /fixboot
bootrec /rebuildbcd
```

**方法 3：重建 BCD**
```
bcdboot C:\Windows /s C: /f UEFI
```

## 数据恢复

### 误删文件怎么恢复？

误删文件后**立刻停止使用那个硬盘**，写入新数据会覆盖被删文件。

**恢复方法：**

1. **回收站**：最简单，直接还原
2. **文件历史记录**：如果开了这个功能，可以恢复以前的版本
3. **数据恢复软件**：
   - Recuva（免费）：简单易用
   - DiskGenius（免费版够用）：国产，功能强
   - R-Studio（付费）：专业级，恢复率高

**重要提示：**
- 数据恢复软件不要装在要恢复的硬盘上
- 恢复的文件保存到其他硬盘
- 机械硬盘误删后恢复率高，固态硬盘低（TRIM 会清理）

### 硬盘数据怎么备份？

**备份策略 3-2-1 原则：**
- **3** 份数据副本
- **2** 种不同存储介质
- **1** 份异地备份

**备份方法：**
- **系统备份**：用 Windows 自带的系统映像备份或第三方工具（如傲梅）
- **文件备份**：手动复制到移动硬盘，或用同步软件
- **云备份**：OneDrive、百度网盘、阿里云盘
