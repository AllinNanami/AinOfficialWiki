# 硬件选购

买电脑配件，不是越贵越好，关键是搭配均衡。CPU 和显卡要匹配，显示器要验清楚，每个环节都有坑。这篇指南帮你把钱花在刀刃上。

4. **新装机**：建议上新平台（Intel 13 代 / AMD 7000 系），老平台以后升级空间小

### CPU 和显卡怎么搭配才不浪费？

**搭配原则：** 显卡预算通常是整机预算的 30-40%。

| 预算区间 | CPU 推荐 | 显卡推荐 |
|---------|---------|---------|
| 3000-4000 元 | i5-12400F / R5 5600 | RTX 4060 / RX 7600 |
| 5000-6000 元 | i5-13600KF / R5 7600X | RTX 4060 Ti / RX 7700 XT |
| 7000-8000 元 | i5-13600KF / R7 7700X | RTX 4070 / RX 7800 XT |
| 10000-12000 元 | i7-13700K / R7 7800X3D | RTX 4070 Ti Super |
| 15000 元以上 | i9-13900K / R9 7950X | RTX 4080 Super / 4090 |

**什么情况会 CPU 瓶颈？**
- 1080P 低画质高帧率游戏：CPU 跟不上显卡的输出速度
- 策略游戏、模拟游戏：大量 AI 计算吃 CPU
- 网游多开：每个窗口都要 CPU 处理

**什么情况会显卡瓶颈？**
- 4K 高画质游戏：显卡渲染不过来
- 光追游戏：光追计算全靠显卡
- AI 绘图、深度学习：显存和算力是关键

## 内存选购

### DDR4 vs DDR5，现在怎么选？

**2025 年建议：新装机直接上 DDR5。**

| 对比项 | DDR4 | DDR5 |
|--------|------|------|
| 价格 | 便宜 | 已经降下来了 |
| 性能 | 够用 | 更好，尤其是 1% low 帧 |
| 升级空间 | 到头了 | 未来主流 |
| 兼容性 | 老平台 | 新平台 |

**内存容量怎么选？**

| 使用场景 | 推荐容量 |
|---------|---------|
| 办公、上网、看视频 | 8GB（勉强够） |
| 主流游戏、日常使用 | 16GB（推荐） |
| 高端游戏、轻度创作 | 32GB（够用） |
| 视频剪辑、3D 建模 | 32-64GB |
| AI 训练、专业渲染 | 64GB+ |

**内存频率怎么选？**

- DDR4：3200MHz 或 3600MHz，性价比最高
- DDR5：6000MHz 是甜点，AMD 平台最佳频率
- 时序 CL 值越低越好，但价格也越贵

**品牌推荐：**
- 高端：芝奇幻锋戟、海盗船复仇者
- 主流：金士顿 FURY、英睿达
- 性价比：光威、阿斯加特

**一定要买两根组双通道！** 单通道和双通道性能差距 10-20%，尤其是游戏帧数。

## 主板选购

### 主板选购要点

主板是最容易被忽略的配件，但它决定了整机的扩展性和稳定性。

**选购清单：**

1. **芯片组**：决定支持的 CPU 和功能，跟 CPU 配套选
2. **供电相数**：高端 CPU 需要好的供电，入门 CPU 不用太在意
3. **内存插槽**：最大支持容量、最高频率、是否支持 XMP/EXPO
4. **M.2 接口**：数量、位置、是否支持 PCIe 5.0
5. **PCIe 插槽**：显卡槽是 x16 还是 x8，有没有额外的 x1 槽
6. **USB 接口**：后面板和前面板各多少个、什么版本
7. **网络**：千兆还是 2.5G 网口、有没有 Wi-Fi
8. **音频**：声卡芯片型号、有没有独立音频区域
9. **I/O 挡板**：一体式还是分体式（一体式安装方便）
10. **BIOS 功能**：Flash BIOS 按钮（不用 CPU 也能更新 BIOS）

**品牌选择：**

| 品牌 | 特点 | 代表系列 |
|------|------|---------|
| 华硕 | 用料好、BIOS 好用、售后好 | ROG、TUF、PRIME |
| 微星 | 性价比高、功能全 | MEG、MAG、PRO |
| 技嘉 | 供电堆料足 | AORUS、EAGLE、UD |
| 华擎 | 价格便宜、功能够用 | Steel Legend、Pro RS |

## 电源选购

### 电源怎么选？别在电源上省钱

电源是整机的供电核心，选不好可能烧硬件。电源炸了可能带走主板、显卡、硬盘，损失比电源贵多了。

**功率计算：**

简单公式：CPU TDP + 显卡 TDP + 150W 余量

| 配置 | 推荐电源功率 |
|------|------------|
| i5 + RTX 4060 | 550W |
| i5 + RTX 4070 | 650W |
| i7 + RTX 4070 Ti | 750W |
| i7 + RTX 4080 | 850W |
| i9 + RTX 4090 | 1000W |

**认证等级：**

| 等级 | 转换效率 | 价格 | 建议 |
|------|---------|------|------|
| 白牌 | 80% | 便宜 | 不推荐 |
| 铜牌 | 85% | 适中 | 预算有限可以选 |
| 金牌 | 90% | 主流 | 推荐 |
| 白金 | 92% | 贵 | 高端平台 |
| 钛金 | 94% | 很贵 | 发烧级 |

**模组类型：**
- **全模组**：线材全可拆，走线方便，推荐
- **半模组**：主板线固定，其他线可拆，够用
- **非模组**：线材全固定，便宜但走线麻烦

**品牌推荐：**
- 高端：海韵、振华、海盗船
- 主流：酷冷至尊、安钛克、鑫谷
- 性价比：长城、航嘉

**ATX 3.0 是什么？**

ATX 3.0 是新的电源标准，原生支持 12VHPWR 显卡供电接口，不需要转接线。RTX 4070 Ti 及以上的显卡建议选 ATX 3.0 电源，转接线烧接口的案例不少。

## 机箱选购

### 机箱怎么选？

机箱不只是装硬件的盒子，它影响散热、走线、扩展性和美观。

**选购要点：**

1. **板型兼容**：确认支持你的主板大小（ATX / M-ATX / ITX）
2. **显卡限长**：高端显卡很长，确认机箱能装下
3. **散热器限高**：塔式风冷很高，确认机箱空间够
4. **风道设计**：前进后出、下进上出是主流风道
5. **硬盘位**：需要装机械硬盘的话看有没有硬盘笼
6. **前面板接口**：USB-C 有没有、几个 USB-A
7. **理线空间**：背面理线空间够不够，有没有理线槽
8. **防尘网**：进风口有没有防尘网，好清理

**机箱大小怎么选？**

| 板型 | 推荐机箱 | 优点 | 缺点 |
|------|---------|------|------|
| ITX | 迷你机箱 | 体积小、好看 | 扩展性差、装机难度高 |
| M-ATX | 中小机箱 | 性价比高、够用 | 扩展性一般 |
| ATX | 中塔机箱 | 扩展性好、兼容性强 | 体积大 |
| E-ATX | 全塔机箱 | 最强扩展 | 很占地方 |

**风道建议：**
- 前面板进风（2-3 个风扇）
- 后面板出风（1 个风扇）
- 顶部出风（2-3 个风扇，或装水冷排）
- 底部进风（电源独立风道）

风扇不是越多越好，多了反而互相干扰。3 进 2 出或 3 进 3 出就够用了。

**关键参数：**

| 参数 | 说明 | 建议 |
|------|------|------|

## 验货注意事项

### 各配件验货要点

**CPU：**
- 检查包装是否完好，封条有没有撕开
- 检查针脚/触点有没有损坏
- 对照型号和包装是否一致

**内存：**
- 检查金手指有没有划痕
- 确认容量和频率跟包装一致
- 上机后用 CPU-Z 确认颗粒信息

**固态硬盘：**
- 用 CrystalDiskInfo 检查通电时间和写入量（应该是 0 或很小）
- 用 CrystalDiskMark 跑速度，确认达标
- 检查容量是否正确

**显卡：**
- 检查 PCB 有没有损坏
- 确认型号标签和实际一致
- 上机跑 3DMark 确认性能正常

**显示器：**
- 检查坏点：纯色背景下看有没有亮点或暗点
- 检查漏光：黑色背景下看边缘有没有漏光
- 检查刷新率：UFO Test 网站确认是否真高刷
- 检查色差：用校色仪或肉眼对比

**电源：**
- 检查线材有没有破损
- 确认功率和认证等级跟包装一致
- 有条件用功率计测一下实际输出

---

以上就是硬件选购的全部建议。记住一个原则：**均衡搭配，别在某一个配件上堆太多钱，也别在某一个配件上省太多钱。** CPU 和显卡是核心，电源和散热别省，内存和硬盘够用就行。
