# 装机指南

自己组装电脑，从选购配件到点亮开机，整个流程其实没想象中那么复杂。关键是搞清楚步骤，别跳步。这篇指南把装机、装系统、装驱动、烤机测试每个环节都讲清楚。

## 装机前准备

### 工具清单

装机前准备好这些工具，别装到一半发现缺东西：

| 工具 | 用途 | 必要性 |
|------|------|--------|
| 十字螺丝刀 | 拧螺丝 | 必备 |
| 磁性螺丝刀 | 吸螺丝方便操作 | 推荐 |
| 扎带 / 魔术贴 | 走线理线 | 推荐 |
| 剪刀 | 剪扎带和包装 | 必备 |
| 防静电手环 | 防止静电损坏硬件 | 建议有 |
| 手电筒 | 照暗处接口 | 有就更好 |
| 硅脂 | 涂 CPU 散热器 | 散热器通常自带 |

### 配件到手后先验货

别急着装，先检查配件有没有问题：

1. **CPU**：检查针脚有没有弯（Intel 看主板触点，AMD 看 CPU 针脚）
2. **内存**：检查金手指有没有划痕
3. **固态硬盘**：检查接口有没有损坏
4. **主板**：检查板面有没有划痕、电容有没有歪
5. **显卡**：检查 PCB 有没有裂痕、接口有没有损坏
6. **电源**：检查线材有没有破损

有问题直接联系商家退换，装上去再发现就不好说了。

### 装机步骤详细拆解

怕你看视频记不住，这里把关键步骤列出来：

**第一步：安装 CPU**

1. 打开主板 CPU 插槽的压杆
2. 对准 CPU 和插槽的三角标记（金色三角对准插槽三角）
3. 轻放 CPU，不要用力按，靠重力自然落下
4. 压下压杆扣紧

**注意：**
- Intel 12/13/14 代是 LGA 1700 插槽，针脚在主板上，CPU 是触点
- AMD 锐龙 7000 系是 AM5 插槽，同样是 LGA，针脚在主板上
- 老款 AMD（AM4）针脚在 CPU 上，要更小心别弯针

**第二步：安装内存**

1. 打开内存插槽两端的卡扣
2. 对准内存缺口和插槽（防呆设计，不对插不进去）
3. 均匀用力按下，听到"咔嗒"一声卡扣自动扣上

**注意：**
- 双通道要插 2、4 槽（从 CPU 往外数），不是相邻的槽
- 只有一根内存时插第 2 槽（离 CPU 近的那个）
- 用力要均匀，别只按一头

**第三步：安装 M.2 固态硬盘**

1. 拆下主板 M.2 散热马甲（有的话）
2. 将固态硬盘斜着插入 M.2 接口
3. 按下固态尾部，用螺丝固定

**注意：**
- 有的主板自带免螺丝卡扣，不需要拧螺丝
- PCIe 4.0/5.0 固态建议装散热马甲，不然温度高会掉速
- 第一个 M.2 通常是直连 CPU 的，速度最快

**第四步：安装散热器**

1. 在 CPU 表面涂硅脂（米粒大小就够）
2. 安装散热器底座和扣具
3. 拧紧螺丝（对角线交替拧，不要一颗拧到底）

**注意：**
- 散热器自带的硅脂通常够用，不用额外涂
- 塔式风冷要注意方向，风扇朝向机箱后部出风
- 水冷排通常装在机箱顶部或前面板

**第五步：安装主板到机箱**

1. 先装主板 I/O 挡板（有的主板自带就不需要）
2. 对准机箱铜柱和主板螺丝孔
3. 拧紧主板螺丝

**注意：**
- 检查机箱铜柱位置和主板孔位是否对应，多余的铜柱要拆掉
- 螺丝不要拧太紧，拧到位就行

**第六步：安装电源**

1. 将电源放入机箱电源仓
2. 拧紧螺丝固定
3. 接好主板供电线（24pin）和 CPU 供电线（8pin）

**注意：**
- 先把电源线穿过机箱背线孔再插
- CPU 供电线在主板左上角，别漏接
- 模组电源先接好线再装进机箱

**第七步：安装显卡**

1. 拆掉机箱对应的 PCIe 挡板
2. 将显卡插入 PCIe x16 插槽，听到"咔嗒"声
3. 接好显卡供电线（6pin/8pin/12VHPWR）

**注意：**
- 显卡很重，建议买个显卡支架防止 PCIe 插槽变形
- 新显卡（RTX 40 系）用 12VHPWR 接口，老电源需要转接线
- 转接线要插紧，松动会烧接口

**第八步：接线**

1. 机箱前面板跳线（开机键、重启键、指示灯、USB、音频）
2. 机箱风扇供电线
3. 硬盘 SATA 数据线和供电线（如果有的话）

**注意：**
- 跳线接法看主板说明书，每块主板位置不同
- 开机键和重启键不分正负极，LED 灯分正负极
- USB 3.0 接口有防呆设计，对准插就行

**第九步：理线**

1. 多余的线材塞到机箱背面
2. 用扎带或魔术贴固定
3. 确保线材不挡住风扇

**理线不是为了好看，是为了散热和以后维护方便。**

### 装机常见问题

**Q：装完按开机键没反应怎么办？**
1. 检查电源开关有没有打开（电源背面的 I/O 开关）
2. 检查主板 24pin 供电和 CPU 8pin 供电有没有插紧
3. 检查机箱开机键跳线有没有接对
4. 检查电源有没有烧（风扇转不转）

**Q：开机风扇转但显示器没画面？**
1. 检查显示器线有没有接对（接显卡，不是主板）
2. 检查显卡有没有插紧、供电有没有接
3. 检查内存有没有插紧（重新拔插一次）
4. 看主板 Debug 灯，对照说明书排查

**Q：开机反复重启？**
1. 检查 CPU 散热器有没有装好（过热保护重启）
2. 检查内存兼容性（换个槽试试）
3. 检查电源功率够不够
4. 进 BIOS 恢复默认设置

**Q：装机时要注意静电吗？**
- 正常环境不太需要担心，摸一下金属机箱放掉静电就行
- 干燥的冬天或地毯环境建议戴防静电手环
- 不要穿毛衣装机

**Q：CPU 针脚弯了怎么办？**
- 小弯可以用针或信用卡轻轻拨正
- 弯太多建议找专业维修，自己掰断就完了
- Intel 平台针脚在主板上，主板比 CPU 便宜，运气好只坏主板

### 系统安装前的 BIOS 设置

装系统前先进 BIOS 做几个关键设置：

1. **开启 XMP/EXPO**：让内存跑到标称频率，不然默认跑 2133/4800MHz
2. **设置启动顺序**：把 U 盘设为第一启动项
3. **确认硬盘模式**：AHCI 模式（现在基本默认就是）
4. **关闭安全启动**（如果装 Linux 或老系统）：Windows 11 需要开启
5. **确认 CSM 设置**：新系统用 UEFI 模式，关闭 CSM

**怎么进 BIOS？**
- 开机时反复按 Del 键（大多数主板）
- 部分主板是 F2 或 F12
- 看主板开机画面提示

### Windows 11 安装注意事项

Windows 11 比 10 多了几个硬件要求：

- **TPM 2.0**：必须开启，在 BIOS 里找 TPM 或 fTPM 选项
- **安全启动**：必须开启
- **4GB 内存**：最低要求，建议 16GB 起步
- **64GB 存储**：系统盘至少 64GB

**绕过 TPM 检查的方法（不推荐但备着）：**
安装时按 Shift+F10 打开命令行，输入 `regedit`，导航到 `HKEY_LOCAL_MACHINE\SYSTEM\Setup`，新建一个 `LabConfig` 项，在里面新建 DWORD 值 `BypassTPMCheck` 设为 1。

### 系统分区方案

**推荐方案：UEFI + GPT**

| 分区 | 大小 | 用途 |
|------|------|------|
| EFI 系统分区 | 512MB | 引导文件 |
| MSR 分区 | 16MB | 系统保留 |
| C 盘（系统盘） | 100-200GB | Windows 系统和软件 |
| D 盘（数据盘） | 剩余空间 | 文件、游戏 |

**为什么用 GPT 不用 MBR？**
- GPT 支持 2TB 以上硬盘
- GPT 支持更多分区
- UEFI 启动更快更安全
- Windows 11 强制要求 GPT

## 驱动安装

### 驱动程序是什么？如何正确安装纯净版驱动？

装完系统第一件事是装驱动。驱动程序是让硬件和系统沟通的桥梁，没装驱动硬件就发挥不了性能。

**驱动安装顺序：**

1. **芯片组驱动**：主板官网下载，让系统正确识别所有硬件
2. **显卡驱动**：NVIDIA/AMD 官网下载最新版
3. **网卡驱动**：如果系统没自动识别，用手机 USB 共享网络先上网
4. **声卡驱动**：主板官网或 Windows Update
5. **其他驱动**：蓝牙、Wi-Fi、USB 等

**去哪里下载驱动？**

| 硬件 | 下载地址 |
|------|---------|
| NVIDIA 显卡 | https://www.nvidia.cn/drivers |
| AMD 显卡 | https://www.amd.com/en/support |
| Intel 芯片组/CPU | https://www.intel.cn/content/www/cn/zh/download-center |
| 主板驱动 | 主板品牌官网（华硕、微星、技嘉等） |

**千万不要用驱动精灵、360 驱动大师这类第三方工具！** 它们会捆绑垃圾软件，装一堆用不上的东西。驱动去官网下，虽然麻烦点但干净。

<BiliBili bvid="BV1v7411e7AE" />

### NVIDIA 显卡驱动安装建议

**Game Ready 驱动 vs Studio 驱动：**
- **Game Ready（GRD）**：针对游戏优化，玩游戏选这个
- **Studio**：针对创意软件优化（Pr、Blender、达芬奇），做设计选这个

**要不要用 GeForce Experience / NVIDIA App？**
- 推荐安装，方便更新驱动和使用 ShadowPlay 录屏
- 如果嫌麻烦，可以只装驱动不装 App

**DDU 是什么？**

Display Driver Uninstaller，专门用来彻底卸载显卡驱动。遇到驱动问题时用它清除干净再重装。从 https://www.wagnardsoft.com/ 下载。

<BiliBili bvid="BV1sD421A74F" />

### 烤机测试详细方案

烤机就是让硬件满载运行一段时间，测试稳定性和散热。新装机一定要烤，不然后面蓝屏死机都不知道是硬件问题还是软件问题。

**CPU 烤机：**

| 软件 | 测试内容 | 时长 | 温度上限 |
|------|---------|------|---------|
| AIDA64 | CPU、FPU、缓存 | 30 分钟 | 90°C |
| Cinebench R23 | 多核渲染 | 跑 10 分钟 | 85°C |
| Prime95 | 极端压力测试 | 30 分钟 | 95°C |
| OCCT | 综合测试 | 30 分钟 | 90°C |

**显卡烤机：**

| 软件 | 测试内容 | 时长 | 温度上限 |
|------|---------|------|---------|
| FurMark | 极端压力 | 15 分钟 | 85°C |
| 3DMark Time Spy | 综合性能 | 跑完看分数 | 80°C |
| Unigine Heaven | 持续负载 | 15 分钟 | 80°C |

**内存测试：**

| 软件 | 测试内容 | 时长 |
|------|---------|------|
| MemTest86 | 内存稳定性 | 跑 1-2 轮 |
| TestMem5 | 快速内存测试 | 跑 3 轮 |

**硬盘测试：**

| 软件 | 测试内容 | 关注指标 |
|------|---------|---------|
| CrystalDiskMark | 顺序和随机读写 | 速度是否达标 |
| CrystalDiskInfo | 硬盘健康状态 | 温度、通电时间、健康度 |
| AS SSD Benchmark | 综合性能 | 分数和延迟 |

### 烤机时关注什么？

1. **温度**：CPU 和显卡满载温度是否正常，过高说明散热没装好
2. **蓝屏/死机**：烤机过程中蓝屏说明硬件不稳定，可能是内存频率太高或电源不够
3. **降频**：温度过高会触发降频保护，用 HWInfo64 查看是否有过热降频
4. **功耗**：用功率计测量整机功耗，确认电源够用

### 烤机结果判断

- **30 分钟无蓝屏、温度正常**：基本稳定
- **蓝屏**：内存问题最常见，降频试试；也可能是电源不够
- **温度过高**：散热器没装好，重新涂硅脂、拧紧螺丝
- **花屏/黑屏**：显卡问题，检查供电和接触

## 装完之后

### 系统优化

装完系统、驱动、烤机都通过后，建议做以下优化：

1. **关闭不需要的开机启动项**：任务管理器 → 启动
2. **调整电源计划**：高性能或卓越性能模式
3. **关闭系统通知**：设置 → 系统 → 通知
4. **清理临时文件**：磁盘清理或 CCleaner
5. **创建系统还原点**：以备不时之需

更多优化内容见 [Windows 优化](/pc-diy/windows-optimization)。

### 软件安装建议

装完系统后安装软件的顺序建议：

1. **浏览器**：Chrome / Edge / Firefox
2. **解压软件**：7-Zip 或 Bandizip
3. **输入法**：系统自带就够，或装搜狗
4. **办公软件**：Office 或 WPS
5. **通讯软件**：微信、QQ
6. **播放器**：PotPlayer 或 VLC
7. **安全软件**：Windows Defender 够用，不用装 360

**软件安装注意事项：**
- 从官网下载，别从下载站下
- 安装时看清楚有没有捆绑软件
- 自定义安装路径，别都装 C 盘
- 安装路径用英文，避免兼容性问题
