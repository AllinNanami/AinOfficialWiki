### Windows 11 必做的优化设置

以下是装完系统后建议立刻做的优化：

**1. 关闭不需要的视觉效果**

系统动画会占用资源，关掉后响应更快：

设置 → 辅助功能 → 视觉效果 → 关闭"动画效果"和"透明效果"

或者：系统属性 → 高级 → 性能设置 → 调整为最佳性能（或自定义保留"平滑屏幕字体边缘"）

**2. 关闭不需要的后台应用**

设置 → 应用 → 已安装的应用 → 点击不需要的应用 → 高级选项 → 后台应用权限 → 从不

**3. 关闭通知和建议**

设置 → 系统 → 通知 → 关闭以下：
- "获取提示和建议"
- "获取使用 Windows 时的提示和建议"
- "显示 Windows 欢迎体验"

设置 → 隐私和安全性 → 常规 → 关闭所有选项

**4. 关闭开机自启动**

Ctrl+Shift+Esc 打开任务管理器 → 启动 → 禁用不需要的启动项

保留：杀毒软件、显卡驱动、输入法
禁用：微信、QQ、各种管家、更新服务

**5. 调整电源计划**

设置 → 系统 → 电源和电池 → 电源模式 → 最佳性能

或者：控制面板 → 电源选项 → 选择"高性能"或"卓越性能"

**6. 关闭系统保留存储**

设置 → 系统 → 存储 → 高级存储设置 → 保留的存储 → 关闭（需要管理员权限）

这个功能会预留 7GB 空间给系统更新，小容量硬盘建议关掉。

**7. 关闭传递优化**

设置 → Windows 更新 → 高级选项 → 传递优化 → 关闭"允许从其他电脑下载"

这个功能会用你的带宽给别人传更新文件，相当于 P2P。

**8. 设置默认应用**

设置 → 应用 → 默认应用 → 把浏览器、视频播放器、图片查看器改成你喜欢的

**9. 关闭搜索索引（如果不用 Windows 搜索）**

搜索框搜"服务" → 找到"Windows Search" → 启动类型改为"禁用"

索引服务会持续占用磁盘和 CPU 资源，如果你习惯直接找文件而不是用搜索，可以关掉。

**10. 清理磁盘空间**

设置 → 系统 → 存储 → 开启"存储感知" → 配置清理计划

或者手动清理：磁盘清理 → 清理系统文件 → 勾选 Windows 更新清理、临时文件

## 账户权限

### Windows 中普通管理员、超级管理员和标准账户都有什么区别？

Windows 里有三种账户类型：

| 账户类型 | 权限 | 适用场景 |
|---------|------|---------|
| 标准用户 | 最低，需要输入密码才能安装软件 | 给小孩或老人用 |
| 管理员（普通） | 可以安装软件、改设置，但有 UAC 弹窗 | 日常使用推荐 |
| Administrator（超级管理员） | 最高权限，无 UAC 弹窗 | 不建议日常使用 |

**为什么建议用普通管理员，不用 Administrator？**

1. UAC 弹窗是安全保护，能防止软件偷偷修改系统
2. Administrator 权限太高，误操作可能搞坏系统
3. 中了病毒，Administrator 权限下病毒能做更多坏事

**怎么查看当前账户类型？**

设置 → 账户 → 你的信息 → 看"管理员"还是"标准用户"

### UAC（用户账户控制）怎么设置？

UAC 就是安装软件时弹出的那个"是否允许此应用对你的设备进行更改"的提示。

设置方法：控制面板 → 用户账户 → 更改用户账户控制设置

| 级别 | 说明 | 建议 |
|------|------|------|
| 始终通知 | 最安全，任何更改都通知 | 安全意识强的人 |
| 默认 | 应用尝试更改时通知 | 推荐 |
| 不通知 | 不推荐 | 不安全 |

建议保持默认级别，嫌烦可以调低一格，但别关掉。

## 磁盘管理

### C 盘空间不够怎么办？

C 盘满了系统会变卡，甚至无法更新。以下是清理方法：

**1. 磁盘清理工具**
- 右键 C 盘 → 属性 → 磁盘清理 → 清理系统文件
- 勾选：Windows 更新清理、临时文件、回收站
- 这个最安全，不会误删

**2. 存储感知**
- 设置 → 系统 → 存储 → 开启存储感知
- 配置自动清理计划（每天/每周/每月）
- 自动清理临时文件和回收站

**3. 迁移用户文件夹**
- 把文档、下载、桌面等文件夹移到其他盘
- 右键文件夹 → 属性 → 位置 → 移动

**4. 迁移软件安装路径**
- 新装软件时自定义安装路径到 D 盘
- 已经装在 C 盘的软件，部分可以迁移（用 Steam Mover 或手动）

**5. 关闭休眠功能**
```
powercfg -h off
```
可以释放几个 GB 空间（等于内存大小）

**6. 压缩系统文件**
- CompactOS 是 Windows 10/11 的功能，可以压缩系统文件节省空间
- 命令：`compact /compactos:always`
- 能节省 2-3GB，对性能影响很小

### 新硬盘怎么初始化和分区？

新买的硬盘需要初始化才能使用：

1. 按 Win + X → 磁盘管理
2. 新硬盘会弹出初始化对话框
3. 选择 GPT（推荐）或 MBR
4. 右键未分配空间 → 新建简单卷 → 按提示操作

**分区建议：**
- 系统盘（C 盘）：100-200GB，装系统和软件
- 数据盘（D 盘）：剩余空间，装游戏和文件
- 如果是 500GB 以下的硬盘，建议不分区，全给 C 盘

## 网络优化

### 网速慢怎么排查？

网速慢不一定是网络的问题，可能是系统设置的问题：

**1. 检查网卡驱动**
- 设备管理器 → 网络适配器 → 右键网卡 → 更新驱动
- 去主板官网下载最新的网卡驱动

**2. 关闭 Windows 更新占用带宽**
- 设置 → Windows 更新 → 高级选项 → 传递优化 → 关闭
- 或者限制下载带宽

**3. 修改 DNS**
- 推荐 DNS：
  - 阿里：223.5.5.5 / 223.6.6.6
  - 腾讯：119.29.29.29
  - Cloudflare：1.1.1.1 / 1.0.0.1
- 设置方法：网络设置 → 更改适配器选项 → 右键属性 → IPv4 → 手动设置 DNS

**4. 关闭流量限制**
- 设置 → 网络和 Internet → 数据使用量 → 关闭流量上限

**5. 重置网络**
- 设置 → 网络和 Internet → 高级网络设置 → 网络重置
- 或者命令行：`netsh winsock reset` 然后重启

## 安全设置

### Windows Defender 够用吗？

Windows Defender（Windows 安全中心）是 Windows 自带的杀毒软件，2025 年了，它已经足够强了：

**优点：**
- 免费、无广告、无弹窗
- 系统集成好，资源占用低
- 病毒库更新及时
- 误报率低

**缺点：**
- 偶尔误删文件
- 高级功能需要企业版
- 病毒查杀率略低于专业杀毒

**建议：** 日常使用 Windows Defender 够了，不用装第三方杀毒。如果经常下载破解软件，可以装一个 Malwarebytes（免费版）做补充扫描。

**不要装的东西：**
- 360 安全卫士（弹窗多、捆绑多）
- 2345 全家桶（广告多）
- 各种管家、助手（资源占用高）

### 防火墙怎么设置？

Windows 防火墙默认开启就好，不需要额外设置：

- 公共网络：开启防火墙，阻止所有不必要的连接
- 家庭网络：防火墙开启，但允许文件和打印机共享

**怎么检查防火墙状态？**
- 设置 → 隐私和安全性 → Windows 安全中心 → 防火墙和网络保护

## 常见问题

### 系统变卡了怎么办？

**1. 检查启动项**
- 任务管理器 → 启动 → 禁用不需要的
- 很多软件默认开机自启，白白占用资源

**2. 检查后台进程**
- 任务管理器 → 进程 → 按 CPU 或内存排序
- 找到占用高的进程，看是不是有问题

**3. 检查硬盘健康**
- 用 CrystalDiskInfo 检查硬盘健康状态
- 如果显示"警告"或"危险"，备份数据换硬盘

**4. 检查温度**
- 用 HWInfo64 查看 CPU 和显卡温度
- 温度过高会降频，导致变卡
- 清灰、换硅脂、改善散热

**5. 重置系统**
- 设置 → 系统 → 恢复 → 重置此电脑
- 保留文件：只重装系统，保留个人文件
- 删除所有：彻底重装，啥都不留
- 这是最后的手段，先试其他方法

### 蓝屏了怎么办？

蓝屏（BSOD）是系统遇到严重错误时的保护机制。

**1. 记录错误代码**
- 蓝屏时会显示错误代码（如 CRITICAL_PROCESS_DIED）
- 拍照记下来，搜索解决方案

**2. 常见蓝屏原因和解决：**

| 错误代码 | 原因 | 解决方法 |
|---------|------|---------|
| CRITICAL_PROCESS_DIED | 系统进程崩溃 | 重装系统或修复启动 |
| MEMORY_MANAGEMENT | 内存问题 | 重新插拔内存，跑 MemTest |
| IRQL_NOT_LESS_OR_EQUAL | 驱动问题 | 更新或回退驱动 |
| PAGE_FAULT_IN_NONPAGED_AREA | 内存或硬盘问题 | 检查内存和硬盘 |
| KERNEL_DATA_INPAGE_ERROR | 硬盘问题 | 检查硬盘健康，换数据线 |
| SYSTEM_SERVICE_EXCEPTION | 驱动或系统文件损坏 | 更新驱动，SFC 修复 |

**3. 通用排查步骤：**
1. 最近装了什么软件或驱动？卸载试试
2. 跑 MemTest86 检查内存
3. 跑 CrystalDiskInfo 检查硬盘
4. 更新显卡驱动
5. 运行 `sfc /scannow` 修复系统文件
6. 运行 `DISM /Online /Cleanup-Image /RestoreHealth` 修复系统映像
7. 最后手段：重装系统

### 开机慢怎么优化？

**1. 检查启动项**
- 任务管理器 → 启动 → 禁用不需要的
- 影响大的：各种云盘、通讯软件、更新服务

**2. 检查是否开了快速启动**
- 控制面板 → 电源选项 → 选择电源按钮的功能 → 启用快速启动
- 快速启动能加快开机速度，但可能导致一些问题（如无法访问 BIOS）
- 如果有问题可以关掉

**3. 检查硬盘**
- 如果系统装在机械硬盘上，开机肯定慢
- 换成固态硬盘，开机速度提升巨大（从 1 分钟变成 10 秒）

**4. 减少开机服务**
- Win + R 输入 `msconfig`
- 服务选项卡 → 勾选"隐藏所有 Microsoft 服务" → 禁用不需要的
- 谨慎操作，不知道是什么的服务别禁用

---

以上就是 Windows 优化的全部内容。优化不是一次性的事，系统用久了还是会变脏，建议每隔几个月清理一次。实在不行就重装系统，比重装快的是格式化重装。
